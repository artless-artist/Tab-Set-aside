# Tab-Set-aside
* 旧版Edge浏览器“搁置标签页”功能在火狐浏览器上的复刻
* 由于火狐扩展商店的合规需求，仅适用于火狐浏览器140+的版本
## 功能
* 将想要稍后再看的标签页搁置起来，被搁置的标签页显示在扩展弹窗的列表中，单击即可打开
## 使用方法
* 单击地址栏上的图标（对于about开头的标签页与新建标签页无效）以搁置当前的标签页
* 也可以在扩展弹窗中单击“搁置当前窗口的所有标签页”一次搁置所有标签页
## 特性
* 支持火狐浏览器的身份标签页区分，下次打开会使用对应的身份容器
* 在“搁置当前窗口的所有标签页”时，这些标签页会显示在一个“工作区”中，可以对工作区重命名
* 支持搁置后关闭、打开后移除的设置
* 支持标签页按添加时间顺序/倒序排列的设置
